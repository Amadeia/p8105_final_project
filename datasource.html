Data Description

  • The Google Trends dataset on Midterm Election results is from the Google Trends Datastore. These datasets are created by the News Lab at Google team. We used the “Top searched issues by US Congressional District, 09/26/2018.” This data only covers searches in the U.S. We chose the dataset from 9/26/2018 because our goal was to examine search trends prior to the Midterm Election. This dataset provides a search index for the most highly searched terms. To obtain the search index, each data point is divided by the total searches of a specific geography and time range. Resulting numbers are scaled from 0 to 100. Google Trends adjusts the search data so results are proportionate to the specific time and location of a query. More information can be found at Trends Help

  • The NY Times Election Results data is taken from the New York Times report of Midterm Election results. It provides data on the candidate and their respective party, number of votes, and percentage of votes. Additionally, it provides voting data by county.

  • We used the Cook Partisan Voting Index (PVI) to determine the most Republican and Democratic districts. This value was determined by comparing each district’s prior two presidental elections to the nation as a whole. We used this in our exploratory analysis of the most searched topics by party.
